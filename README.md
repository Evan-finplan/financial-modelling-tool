# Financial Modelling Tool (AU Super + Tax + CGT)

A Streamlit-based financial modelling tool for Australian retirement planning.

## Features

- Deterministic projection
- Monte Carlo simulation
- Dual-person modelling
- Super accumulation → pension
- TBC enforcement
- CGT modelling
- Adviser / Client view
- Excel export
- Editable assumption presets

## Run locally

pip install -r requirements.txt
streamlit run app.py

## Deployment

Deploy via Streamlit Community Cloud using:
- Repo
- Branch: main
- Entry file: app.py

## Disclaimer

This tool is for modelling and educational purposes only.
